[ccc - Cloud Cost Checker](https://github.com/kunitsuinc/ccc) collects, calculates, graphs and notifies IaaS costs.  

## Project Goal

- Inform you of IaaS costs so that you are aware of sudden cost increases

## Supported

### IaaS

- Google Cloud Platform

### Notification

- via Slack
